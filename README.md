# Magic Contability - AI Company Investment Tracker

A comprehensive visualization of real AI company investment relationships using AI-powered analysis and Mermaid graphs. All data is based on verified public information from 2022-2025.

## Overview

This repository tracks and visualizes investment relationships between major tech companies and AI startups. The data includes real investment amounts, acquisitions, and valuations in billions of USD, presented using interactive Mermaid diagrams.
